# E-Voting-DAPP-using-Blockchain


#### Requirements
- Ethereum enabled browser or Metamask installed (https://metamask.io/)
- node installed (https://nodejs.org/en/download/)
---

#### Installation

```
$ npm install
$ node index.js
```
